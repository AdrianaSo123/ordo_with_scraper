# Sprint 19: Contextual Mentions (@ / [[ / #)

## Goal

Enable "speed of thought" referencing of the book library via autocompleting mentions.

## Features

- **@Mentions**: Autocomplete for practitioners (Dieter Rams, etc.).
- **[[ Mentions**: Autocomplete for chapters.
- **# Mentions**: Autocomplete for frameworks and checklists.
- **UI Popup**: A sleek overlay that appears when triggers are typed in the textarea.

## Success Criteria

- Typing `@D` shows a list of practitioners starting with D.
- Selection inserts a formatted tag or link into the chat text.
- The AI recognizes these as specific context identifiers.
